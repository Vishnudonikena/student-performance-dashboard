## What This Project Does

This dashboard analyses student academic performance and placement outcomes across key variables — study hours, gender, and marks — to answer one practical question: **what factors actually predict whether a student gets placed?**

Built entirely in Power BI with Excel as the data source. No fluff, just findings a college placement cell or HR team could act on.

---

## Key Findings

| Finding | Detail |
|---------|--------|
| 📚 Study hours threshold | Students studying **5+ hours/day** show measurably higher marks and placement rates |
| 👩‍🎓 Gender performance gap | Female students outperform male students in total marks on average |
| 🎯 Marks → Placement link | Strong positive correlation between total marks and placement probability |

> These aren't just observations — the dashboard lets you slice each one interactively by cohort, gender, and score range.

## How to Open

1. Download `student-performance-dashboard.pbix`
2. Open in [Power BI Desktop](https://powerbi.microsoft.com/desktop) (free)
3. All data is embedded — no external connections needed
4. Use the slicers on each page to explore by gender, study hours, and score range

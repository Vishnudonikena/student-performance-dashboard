# 🎓 Student Performance & Placement Dashboard

## 📊 Project Overview
This project analyzes student performance based on study hours, gender, and placement status using Power BI.

## 🔍 Key Insights
- Students studying more than 5 hours achieve higher marks and better placement chances  
- Female students slightly outperform male students in total marks  
- Higher marks strongly increase placement probability  

## 🛠️ Tools Used
- Power BI  
- Excel  
- Data Visualization  

## 📁 Files Included
- student-performance-dashboard.pbix  
- Dashboard image  
- Dataset  

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Explore the dashboard
